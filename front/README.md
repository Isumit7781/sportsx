# Sports Management System Landing Page

A responsive landing page for a Sports Management System built with Bootstrap and Tailwind CSS.

## Features

- Fully responsive design that works on all devices
- Built with Bootstrap 5 and Tailwind CSS
- Modern, clean UI with smooth animations
- Optimized for performance
- Cross-browser compatible

## Sections

1. **Navigation** - Responsive navbar with mobile menu
2. **Hero** - Eye-catching introduction with call-to-action buttons
3. **Features** - Highlights of the platform's capabilities
4. **Statistics** - Key metrics and numbers
5. **Testimonials** - Customer reviews and feedback
6. **Pricing** - Transparent pricing plans
7. **Call to Action** - Encouragement to sign up
8. **Footer** - Contact information and additional links

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Tailwind CSS
- Font Awesome
- Google Fonts

## Getting Started

Simply open the `index.html` file in your web browser to view the landing page.

## Customization

- Update the placeholder images with actual sports-related images
- Modify the color scheme in the CSS file
- Add or remove sections as needed
- Update the content to match your specific sports management system
